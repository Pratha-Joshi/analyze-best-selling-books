# Amazon Best-Selling Books Analysis

This project analyzes Amazon’s Top 50 Best-Selling Books dataset (2009–2019) using Python and Pandas.  
It focuses on exploring popular authors, book categories, and basic trends in the dataset.

## Objective
- Explore Amazon bestseller data using Pandas
- Identify top authors and book categories
- Practice fundamental data analysis workflows

## Tools & Technologies
- Python
- Pandas
- Jupyter Notebook

## Dataset
**Amazon Top 50 Best-Selling Books (2009–2019)**  
The dataset contains information such as book name, author, genre, user rating, number of reviews, price, and year.

## Key Analysis Performed
- Counted top-selling authors
- Analyzed distribution of Fiction vs Non-Fiction books
- Explored basic trends in ratings and reviews
- Exported summary results to CSV files

## Key Insights
- A small number of authors appear repeatedly among top bestsellers
- Fiction books dominate the bestseller list
- User ratings remain consistently high across years

## How to Run
This notebook was originally developed on Kaggle.  
If running locally, dataset file paths may need to be updated.
